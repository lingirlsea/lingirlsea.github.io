- 添加 > $ git add > - filename 将指定文件添加到跟踪或提交到暂存区 - 提交 > $ git commit > - 不加参数，默认启用文本编辑器以便输入提交信息 > - -m, --message <message> 将暂存区的文件提交至本地Git仓库 > - -a -m 跳过 git add，自动把所有已经跟踪过的文件暂存起来一并提交 - 状态 > $ git status > > -s, --short 状态简览 > - 新添加的未跟踪文件前面有 ?? 标记 > - 新添加到暂存区中的文件前面有 A 标记 > - 修改过的文件前面有 M 标记，右边的 M 表示该文件被修改了但是还没放入暂存区，左边的 M 表示该文件被修改了并放入了暂存区
